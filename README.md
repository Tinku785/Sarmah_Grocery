# Sarmah Grocery 🛒

A premium, modern grocery shopping web experience built with React and Vite. This project showcases a high-end user interface with dynamic 3D renders, smooth scroll animations, and a sleek dark mode aesthetic.

## ✨ Features

- **Premium UI/UX:** Frosted glassmorphism design with a vibrant color palette tailored for a high-end feel.
- **Dynamic Categories:** Clickable expanding bento-box categories ("Chilled & Frozen", "Kitchen Staples", "Daily Fresh", and "Snacks") featuring pop-out 3D product renders.
- **Micro-Animations:** Fluid entrance effects, hover states, and buttery-smooth page scrolling.
- **Interactive Sticky Controls:** Easily accessible floating buy/cart buttons optimized for both desktop and mobile viewports.
- **Responsive Layout:** Perfectly scales text, headings, and imagery across all devices, from mobile to ultra-wide desktop displays.

## 🛠️ Tech Stack

- **Framework:** [React 18](https://reactjs.org/) powered by [Vite](https://vitejs.dev/) for lightning-fast HMR and building.
- **Styling:** [Tailwind CSS 3](https://tailwindcss.com/) bundled with the `tailwind-scrollbar` plugin for custom scrolling.
- **Animations:** [Framer Motion](https://www.framer.com/motion/) and [GSAP](https://gsap.com/) for complex, smooth visual transitions.
- **Smooth Scrolling:** [Lenis](https://lenis.studiofreight.com/) for fluid, momentum-based scrolling.
- **Icons:** [Lucide React](https://lucide.dev/) for sleek, modern SVG iconography.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You will need [Node.js](https://nodejs.org/en/download/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Tinku785/Sarmah_Grocery.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Sarmah_Grocery
   ```
3. Install the NPM dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit the local address provided by Vite (usually `http://localhost:5173`).
